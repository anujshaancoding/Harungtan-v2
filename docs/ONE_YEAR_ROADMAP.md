## Phase 1: Foundation (Month 1-2)

### Month 1: Brand Setup & Supplier Discovery

**Week 1-2: Legal & Administrative**
- [ ] Register GST (online, free, 7-10 days processing)
- [ ] Register MSME/Udyam registration (online, free, instant)
- [ ] File trademark application for "Harungtan" (Class 25 - Clothing) -- Rs.4,500
- [ ] Open a current account (Razorpay X / HDFC / ICICI business account)
- [ ] Set up accounting (Zoho Books free tier or simple spreadsheet)

**Week 1-2: Brand Identity**
- [ ] Finalize logo design (hire freelancer on Fiverr/local -- Rs.2,000-5,000)
- [ ] Define brand color palette, typography, and visual guidelines
- [ ] Create brand assets: logo variations, social media templates, email headers
- [ ] Write brand story, about page copy, and product description templates

**Week 2-3: Supplier Sourcing**
- [ ] Research manufacturers on IndiaMart (filter: Tirupur, MOQ < 100, cotton t-shirts)
- [ ] Contact 15-20 manufacturers via call/WhatsApp
- [ ] Request fabric swatches and pricing from 8-10 shortlisted manufacturers
- [ ] Order samples from top 5 manufacturers (budget: Rs.3,000-5,000)

**Week 3-4: Product Planning**
- [ ] Design first collection: 5-6 designs (2 classic, 2 oversized, 1 graphic, 1 acid wash)
- [ ] Define size chart (reference Bewakoof/Souled Store sizing, add 5% tolerance)
- [ ] Select colors: start with safe colors (Black, White, Navy, Olive, Charcoal)
- [ ] Create tech packs (design specifications) for manufacturer

**Deliverables:**
- GST number active
- Trademark application filed
- 5 manufacturer samples received and evaluated
- Brand identity kit ready
- First collection designed

**Budget: Rs.15,000-25,000**

### Month 2: Website Launch & First Inventory

**Week 1-2: Website Finalization**

- [ ] Order fabric labels and care instruction tags
- [ ] Set up Shiprocket account and courier partnerships

**Week 4: Pre-Launch Preparation**

- [ ] Post 9-12 initial posts (brand story, behind-the-scenes, product teasers)
- [ ] Set up WhatsApp Business account
- [ ] Prepare launch email template
- [ ] Create "Coming Soon" landing page variation
- [ ] Test order fulfillment process end-to-end (self-order test)

**Deliverables:**
- Website live with 10-15 product listings

- Packaging materials ready

- Shipping logistics set up

## Decision Checkpoints

### Month 3 Decision: Continue or Pivot?
- If < 10 orders: Revisit product, pricing, or marketing approach
- If 10-30 orders: Proceed as planned, focus on feedback
- If > 30 orders: Accelerate timeline, increase inventory investment

### Month 6 Decision: Scale Investment?
- If revenue < Rs.1L/month: Reduce ad spend, focus on organic and product improvements
- If revenue Rs.1-3L/month: Continue as planned
- If revenue > Rs.3L/month: Consider external funding or larger personal investment

### Month 9 Decision: Year 2 Direction?
- If total revenue < Rs.15L: Stay lean, focus on profitability over growth
- If total revenue Rs.15-30L: Balanced growth, consider first hire
- If total revenue > Rs.30L: Aggressive scaling, explore angel funding (Rs.25-50L)

---

*This roadmap should be reviewed and adjusted monthly based on actual performance. Flexibility is key -- if something works, double down. If something does not, pivot quickly.*
